The Studio

GameWaveStudios

An independent studio building accessible, cross-platform games.

GameWaveStudios is an independent game development studio focused on creating accessible cross-platform games. The studio prioritises optimisation, smooth gameplay, and making games that run well even on lower-end hardware.

Overknight is the studio's flagship project — combining the VelocityVentures FPS mode and ThunderRush racing mode into one continuously evolving, cross-platform build, developed in Godot 3.6 and designed to scale from Android up to Windows PC.
