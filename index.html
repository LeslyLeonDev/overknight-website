About the Game

One project, three ways to play

Overknight is a cross-platform action game developed by GameWaveStudios, combining racing, combat and exploration into a single project. It's built for reach: the game targets both modern hardware and low-end Android devices without cutting the experience down to fit. Development is active and ongoing, with regular updates pushed straight from the Pre-Alpha build.

Features

Built to run everywhere

Every system in Overknight is designed around one rule: it has to run well, even on hardware that struggles with everything else.

⇄Cross-Platform

One project, built to reach players across devices from day one.

📶LAN Multiplayer

Host or join sessions over a local network — no server required.

🌐Online MultiplayerIn Development

Session codes and IP-based matchmaking are already in testing.

🏆Achievements

Track milestones across every mode as the game grows.

⬆Level System

Progress persists across sessions, mode to mode.

◈OverCredits

An in-game currency system tied to future unlocks.

🎮Controller SupportFuture

Full gamepad mapping planned for a future update.

✦Optimised Graphics

A visual style tuned for performance, not just screenshots.

↻Regular Updates

The Pre-Alpha build evolves on a steady, visible cadence.
